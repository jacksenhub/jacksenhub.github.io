我的第一篇博客

这是我用 Hexo + GitHub Pages 搭建的第一篇博客！🎉

为什么要写博客？

  • 记录学习过程，方便以后回顾
  • 锻炼表达能力，把知识讲清楚
  • 打造个人技术品牌，和更多人交流

下一步计划

  1. 学习 Markdown 语法，写出更美观的文章
  2. 美化博客主题，让页面更好看
  3. 每周至少更新一篇技术/生活笔记

保持输出，持续成长！


24年的时候，我尝试过用Word记录下日常生活的想法感悟，其实也就是写日记。后来电脑坏了，重装系统，许多文件也在这时候损坏了，之后我也就没有再记录过。哎，那段时间其实堕落了，每天浑浑噩噩，说到这里，索性回顾一下那段日子吧。

失败从来都有迹可循。从小到大，我一直都在扮演着不属于真正的我的角色，现在想想好累。一直压抑着内心的想法，太压抑了，最终把自己压垮。上大学后，面对各种各样的选择，我害怕去了解，害怕去面对，害怕一切新事物，所以我选择了随波逐流。但这里的随波逐流在现在看，不过是日复一日重复着宿舍、教室、食堂三点一线的枯燥生活。我不知道自己是谁，我到底在干什么，我也从来没问过自己来这里干什么。那时候的我害怕去改变，更害怕去思考。在这样的生活状态下大一过去了，无比平淡。

步入大二，这时期我变得不害怕了，也愿意去尝试新事物了。可我还有许多缺点没有被正视，这也导致我错过了许多机会。在这样的状态下，我的生活反而变糟，我也不知道为什么，只想感谢一切都是最好的安排。

我习惯于“我以为”，而恰恰是这三个字让我的人生停滞不前。我一直在想，很少去做，现在感觉自己真的很愚钝，又或许是认知太低了。人生最重要的课题是认识自己，我却到现在，21岁，经历了痛彻心扉才敢于去正视自己。可失去的那些东西，从来没有第二次机会。

一直以来，那个空虚的自己，用虚无来填补空虚的自己，已经被我杀死了。只是代价太痛，太大。

我太晚才学会做自己，才认识到自己，或许那些机会注定不属于我，但过去不值得沉浸其中，因为脚步要迈向明天。
